# AIRA

> Diseño web 3D 360°, transiciones fuera de serie, minimalismo radical y UX/UI de élite.

---

## ✦ Concepto

AIRA es un sistema de diseño web orientado a experiencias inmersivas: animaciones 3D, transiciones fluidas, layouts minimalistas y una arquitectura de componentes pensada desde la percepción del usuario.

---

## 📁 Estructura del proyecto

```
AIRA/
├── index.html           # Landing principal
├── assets/
│   ├── css/
│   │   ├── tokens.css   # Design tokens (colores, tipografía, espaciado)
│   │   ├── base.css     # Reset y estilos base
│   │   └── main.css     # Estilos globales
│   ├── js/
│   │   ├── main.js      # Lógica principal
│   │   └── animations.js# GSAP & Three.js
│   └── images/          # Assets visuales
├── components/          # Componentes reutilizables
└── pages/               # Páginas adicionales
```

---

## 🛠 Stack tecnológico

| Herramienta | Uso |
|---|---|
| HTML5 / CSS3 | Estructura y estilos |
| GSAP 3.x | Animaciones y ScrollTrigger |
| Three.js | Escenas 3D y WebGL |
| Lenis | Smooth scroll |
| Lucide Icons | Iconografía limpia |

---

## 🎨 Design System

- **Paleta:** Nexus (Beige cálido + Teal primario)
- **Tipografía:** Boska (Display) + Satoshi (Body)
- **Espaciado:** Sistema de 4px
- **Modo:** Light & Dark con toggle
- **Animaciones:** 3D 360°, parallax, morphing, scroll reveals
